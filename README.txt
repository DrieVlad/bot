Постановка 1 задачи.
Создать основной класс, в котором реализовано взаимодействие с ботом. В этом классе создать метод - справка о боте(возвращает описание возможностей бота по ключевым фразам), с поддержкой вызова из любого места программы. Создать первичный скелет вопросов(общих), крутящихся в бесконечном цикле, пока пользователь не выберет во что он хочет поиграть. Также создать метод, в котором пользователь может выбрать в какую игру он хочет играть, из предложенных программой. Добавить возможность выбора. Предусмотреть различные положительные и отрицательные ответы. Создать класс для игры в "Города" для нескольких(одного или более) человек, 
в которой каждый участник в свою очередь называет реально существующий город России, 
название которого начинается на ту букву, которой оканчивается название города предыдущего пользовательзователя или бота.
Постановка 2 задачи. 
Создать вспомогательный класс, реализующий правила игры и исключения с помощью методов 
(выбор города бота, проверка на существование города, фиксирование названных городов, исключение допущения города на редкоиспользуемые буквы). Предусмотреть недопустимые по правилам игры введенные данные участников и сообщить им об этом. Создать класс для игры в "Миллионера". Создать возможность переключения между играми. Реализовать конечный автомат.
